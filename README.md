Приложение 2048 - Плиточки 

Это версия популярной игры 2048, реализованная на HTML, CSS и JavaScript.

Основные особенности:

1. Игровое поле:

   * Сетка 4×4 с плитками, которые отображают числа (2, 4, 8 …).
   * Плитки двигаются по стрелкам или свайпам на мобильных устройствах.
   * Плитки сливаются при совпадении чисел, увеличивая значение и счет.

2. Управление:

   * Десктоп: стрелки на клавиатуре.
   * Мобильные устройства: кнопки и свайпы.
   * Поддержка отмены хода.
   * Кнопка «Начать заново» для сброса игры.

3. Анимации и визуальные эффекты:

   * Плавное движение плиток при ходе.
   * Эффект «pop» при слиянии плиток.
   * Адаптивный дизайн: размеры плиток подстраиваются под экран.

4. Счет и рекорды:

   * Отображение текущего счета.
   * Таблица лидеров с топ-10 рекордов, хранение в localStorage.
   * Возможность сохранять имя игрока при завершении игры.

5. Хранение состояния:

   * Игра сохраняется в localStorage, можно продолжить с того же места.
   * Поддержка восстановления состояния после перезагрузки страницы.

6. Конец игры:

   * Игра заканчивается, когда нет доступных ходов или достигнуто число 2048.
   * Появляется модальное окно с сохранением рекорда и кнопкой перезапуска.
